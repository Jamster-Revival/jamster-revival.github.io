Dancing Sweety

Game Type: Java

Description: Dancing Sweety is a Java rhythm game developed by Mediaplazza Games featuring Sweety the Chick. The gameplay itself is very simple, the player needs to click the phone's keys in the right time in order to fill a Mega Combo bar, when the mega combo bar is filled, the player can view a animation of Sweety along with the text "MEGA COMBO".
